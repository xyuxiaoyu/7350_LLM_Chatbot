How to build your own AI chatbot using the ChatGPT API?

We bring you a tutorial on how to build your own AI chatbot using the ChatGPT API. We have also implemented a Gradio interface, so you can easily demonstrate the AI model and share it with your friends and family.

Things to remember before building an AI chatbot

In this tutorial, from setting up the tools to installing the library, and finally creating an AI chatbot from scratch, we have included all the small details for ordinary users here.

We suggest that you follow the instructions from top to bottom and do not skip any part.

Platform freedom

You can build a ChatGPT chatbot on any platform, whether it's Windows, macOS, Linux, or ChromeOS. The steps for all platforms are almost the same.

Easy to understand

This guide is prepared for ordinary users, and the examples in the instructions are explained very clearly. Therefore, even if you have a rough understanding of computers, you can easily create your own artificial intelligence chatbot.

To create an artificial intelligence chatbot, you don't need a powerful computer with a strong CPU or GPU.

1. Set up the software environment to create an AI chatbot

Before creating an AI chatbot driven by ChatGPT, you need libraries such as Python, Pip, OpenAI and Gradio, an OpenAI API key, and a code editor.

Now, please follow the steps below.

Install Python

First of all, you need to install Python on your computer. Open the link and download the installation file suitable for your platform.

Run the installation file

Next, run the installation file and make sure the checkbox "Add Python.exe to PATH" is enabled. Click "Install Now" and follow the regular steps to install Python.

Check the Python version

Check if Python is installed correctly, open the terminal on your computer and run the following command (Just run one). The terminal will output the Python version. On Linux or some platforms, you may only be able to use the second command.

python --version

python3 --version

Upgrade Pip

Pip is the package manager of Python. Basically, it enables you to install tens of thousands of Python libraries from the terminal. Through Pip, we can install the OpenAI and Gradio libraries.

In the terminal, run the following command to update Pip (Just run one). On Linux or some platforms, you may only be able to use the second command.

python -m pip install -U pip

python3 -m pip3 install -U pip3

Install the OpenAI library

The OpenAI library enables us to interact with ChatGPT through their API.

In the terminal, run the following command to install the OpenAI library (Just run one). If the pip command doesn't work, please try running it with pip3.

pip install openai

pip3 install openai

Install the Gradio library

Gradio enables you to quickly develop a user-friendly web interface so that you can demonstrate your AI chatbot. It also enables you to easily share chatbots on the Internet through a shareable link.

In the terminal, run the following command to install the Gradio library (Just run one). If the pip command doesn't work, please try running it with pip3.

pip install gradio

pip3 install gradio

Download a code editor

Finally, we need a code editor to edit some code.

For Windows, Notepad++ is recommended.

For macOS and Linux, Sublime Text is recommended.

For ChromeOS, Caret is recommended.

If you can adapt to a powerful IDE, you can use VS Code on any platform.

2. Get OpenAI's API key for free

To create an AI chatbot driven by ChatGPT, you need an OpenAI API key. The API key will allow you to call ChatGPT in your own interface and display the results there. OpenAI is offering free API keys, with a free quota worth $5 for the first three months. After the free credit is used up, you will have to pay for API access. You can also use the APIs of other chatbots.

Step 1

Go to https://openai.com/api/, create a free account. If you already have an OpenAI account, just log in.

Step 2

Click on your profile in the upper right corner and select "View API keys" from the drop-down menu.

Step 3

Here, click "Create new secret key" and copy the API key.

Note: You cannot copy or view the entire API key in the future. Therefore, it is strongly recommended to immediately copy and paste the API key into the Notepad file.

Do not publicly share or display the API key. This is a private key and is only used to access your account. You can also delete the API key and create multiple private keys (up to five).

3. Build an AI chatbot using the ChatGPT API and Gradio

It's time to deploy the AI chatbot. We use Gradio to create a simple web interface that can be used both locally and online.

Create the Gradio interface

Open the code editor(like Notepad++) and paste the following code to implement the design of the Gradio interface.

import openai
import gradio as gr

openai.api_key = "Your API key"
messages = [{"role": "system", "content": "You are a helpful and kind AI Assistant."},]

def chatbot(input):
    if input:
        messages.append({"role": "user", "content": input})
        chat = openai.ChatCompletion.create(
            model="gpt-3.5-turbo", messages=messages
        )
        reply = chat.choices[0].message.content
        messages.append({"role": "assistant", "content": reply})
        return reply

inputs = gr.inputs.Textbox(lines=7, label="Chat with AI")
outputs = gr.outputs.Textbox(label="Reply")

gr.Interface(fn=chatbot, inputs=inputs, outputs=outputs, title="AI Chatbot",
            description="Ask anything you want",
            theme="compact").launch(share=True)

Make sure to replace the "Your API key" text with the API key you generated on it yourself. This is the only change you need to make.

Save the code file

Next, click on "File" in the top menu and select "Save as..." from the drop-down menu.

Set the file type

Set the file name to "app.py", and change "Save Type" to "All Types" from the drop-down menu.

Then, save the file to an easily accessible location, such as the desktop. You can change the name according to your own preference, but make sure the .py is attached.

Copy the path

At the location where you save the file (app.py). Right-click it and select "Copy File Address".

Run the Python file

Open the terminal, run the following command, and then click Enter. Please remember that the file paths of your computer will be different.

On Linux or some platforms, you may only be able to use the second command.

python "C:\Users\mearj\Desktop\app.py"

python3 "C:\Users\mearj\Desktop\app.py"

Get the URL

You may receive some warnings, but ignore them.

You can copy the local URL and paste it into the web browser. This is your own artificial intelligence chatbot.

You can also copy the public URL and share it with your friends and family. This link will last for 72 hours, but you still need to keep your computer on because the server instance is running on your computer.

Get your AI chatbot

Your ChatGPT-driven AI chatbot is now live. Now, you can ask any question you want and get the answer in an instant. In addition to alternatives to ChatGPT, you can also use your own chatbot instead of the official website.

To stop the server, move to the terminal and press "Ctrl + C". If it doesn't work, press "Ctrl + C" again.

To restart the AI chatbot server, simply copy the file path again and run the following command once more.

python "C:\Users\mearj\Desktop\app.py"

python3 "C:\Users\mearj\Desktop\app.py"

Please remember that the local URL will be the same, but the public URL will change after each server restart.
