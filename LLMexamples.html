Common Large Language Models

Large language models are important technologies in the field of natural language processing. These models each have their own characteristics and have promoted the rapid development of artificial intelligence technology.

GPT-4.1 Series

From OpenAI

Optimized specifically for tasks like programming, instruction compliance, and long text processing.

  • Programming ability has been significantly enhanced: Scored 54.6% in the SV-Bench Verified test, which is 21.4 percentage points higher than GPT-4o.
  • Enhanced long text processing capability: Supports context Windows of up to 1 million tokens, which is 8 times that of the previous GPT-4o model.
  • Better instruction compliance: Scored 38.3% in the MultiChallenge benchmark test of Scale, which was 10.5 percentage points higher than GPT-4o.
Learn More

Gemini 2.5

From Google DeepMind

Have a context window of one million tokens, adopt the Hybrid Expert (MoE) architecture.

  • Enhance reasoning ability: Excel in mathematics, programming, and logical reasoning tasks, capable of generating functional code or even complete applications.
  • Multimodal understanding: It can analyze text and charts in images and generate relevant descriptions, and is applicable to scenarios such as medical imaging and financial data analysis.
  • Self-Fact-Checking: Reduce hallucination problems and improve the accuracy of answers.
Learn More

Llama 3.1

From MetaAI

It adopts a pure decoder Transformer architecture (non-hybrid expert model), supports a 128K long context window.

  • Performance benchmarking against top closed-source models: The 405B version approaches or surpasses GPT-4o and Claude 3.5 Sonnet in benchmark tests such as MMLU, GSM8K (mathematics), and HumanEval (programming).
  • Multi-language support: Covering 8 languages including English, German, French, Spanish, etc., optimizing the performance of non-English tasks.
  • Tool invocation capability: Built-in search (Wolfram Alpha mathematical reasoning) and custom JSON functionality, suitable for agent development.
Learn More

DeepSeek-R1

From DeepSeek

The ability to conduct in-depth analysis that approaches human "System 2 thinking".

  • Leading in Chinese processing: Outperforms GPT-4o in Chinese SimpleQA and is suitable for localized applications.
  • Strengthen reasoning ability: Outperform Claude 3.5 and GPT-4o in benchmark tests such as mathematics (MATH-500) and programming (HumanEval).
  • Efficient architecture: By adopting multi-head Latent attention (MLA) technology, the KV cache usage is reduced by 93.3%, optimizing the inference speed.
Learn More

Other Large Language Models

The following are other open-source large language models from domestic and foreign companies, research institutions and other organizations. Click on the LLM name to learn more about the model.

Stanford Alpaca

The instructor-tuned LLaMA model is fine-tuned from Meta's large language model LLaMA 7B. Let OpenAI's text-davinci-003 model generate 52K instruction following samples in a self-instruct manner, and use them as the training data for Alpaca.

Dolly

A low-cost LLM that adopts EleutherAI's existing open-source model with 6 billion parameters and makes minor modifications to it to stimulate instruction following capabilities. It demonstrated the same "magical human interaction ability" as that shown by ChatGPT.

Pengcheng • Pangu a

The industry's first pre-trained generative language model with 200 billion parameters centered on Chinese. It supports rich scene applications and performs outstandingly in the fields of text generation such as knowledge question answering, knowledge retrieval, knowledge reasoning, and reading comprehension, and has a strong ability of few-shot learning.

MOSS

An open-source dialogue language model that supports both Chinese and English and multiple plugins. It was pre-trained on approximately 700 billion Chinese, English and code words. Subsequently, through fine-tuning of dialogue instructions, enhanced learning by plugins and training based on human preferences, it has the ability to conduct multiple rounds of dialogues and use various plugins.

BLOOM

A large language model supports 46 natural languages and 13 programming languages. It is easily accessible and can be used to answer questions, translate texts, extract information fragments from files, and also be used to generate code like GitHub Copilot.

StableLM

A language model developed by Stability AI, with 3 billion and 7 billion parameters. It can generate text and code and provide support for a series of downstream applications. It demonstrates how small and efficient models can provide high performance under appropriate training.

Cerebras-GPT

The pre-trained large models in the field of natural language processing open-sourced by Cerebras have a minimum model parameter scale of 111 million and a maximum of 13 billion, with a total of 7 models. It is almost completely open in all aspects with no restrictions. Both the model architecture and the pre-training results are public.

SantaCoder

A language model with 1.1 billion parameters can be used for code generation and completion suggestions for several programming languages such as Python, Java and JavaScript. The basis for training SantaCoder is The Stack (v1.1) dataset.

ChatYuan

A large functional dialogue language model supporting both Chinese and English. Chatyuan-large-v2 is one of the models in the ChatYuan series that achieves high-quality effects through lightweight. Users can perform inference on consumer-grade graphics cards, PCS, and even mobile phones (with a minimum INT4 of only 400M).
